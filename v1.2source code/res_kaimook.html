ร้านไข่มุก

ร้านไข่มุก ร้านอาหารเรือนไม้สัก ที่เปิดมานานกว่า 20 ปี ให้บริการทั้งอาหารไทย และอาหารพื้นเมือง ร้านตกแต่งด้วยเฟอร์นิเจอร์ที่บ่งบอกความเป็นชาวเหนืออย่างแท้จริงภายในร้านบรรยากาศดี ร้านน่านั่ง

รายละเอียด

  • ที่ตั้ง : 23 ถ.ขุนลุมประพาส จองคำ อ.เมือง จ.แม่ฮ่องสอน 58000
  • โทรศัพท์ :053-612-092, 053-612-385
  • เวลาเปิด-ปิด :ช่วงแรก 10.00 - 14.00 น. และช่วงที่ 2 17.00 - 22.00 น.
  • การเดินทาง : ใช้ถนนขุนลุมประพาส จากตัวเมืองแม่ฮ่องสอน สังเกตร้านอยู่ใกล้กับธนาคารกสิกรไทย สาขาแม่ฮ่องสอน
